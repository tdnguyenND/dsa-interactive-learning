# Arrays

Arrays are one of the most fundamental data structures in computer science. This page covers how arrays store elements in contiguous memory, how to access elements by index, and common operations like insertion, deletion, and traversal.

## Overview

An array is a collection of elements stored at contiguous memory locations. Elements can be accessed in constant time using their index, making arrays ideal for random access patterns. However, inserting or deleting elements in the middle requires shifting elements, which can be costly.

arr = [10, 20, 30, 40, 50]

# Access by index - O(1)
print("Element at index 2:", arr[2])

# Append to end - O(1) amortized
arr.append(60)
print("After append:", arr)

# Insert at index - O(n)
arr.insert(1, 15)
print("After insert at index 1:", arr)

# Remove element - O(n)
arr.remove(30)
print("After removing 30:", arr)

# Traverse the array - O(n)
print("All elements:")
for i, val in enumerate(arr):
    print(f"  Index {i}: {val}")`}

## Complexity

| Operation      | Time   | Space |
|----------------|--------|-------|
| Access by index | O(1)  | O(1)  |
| Search          | O(n)  | O(1)  |
| Insert at end   | O(1)* | O(1)  |
| Insert at index | O(n)  | O(1)  |
| Delete          | O(n)  | O(1)  |

*Amortized time complexity.
