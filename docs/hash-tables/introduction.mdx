# Hash Tables

A hash table (or hash map) is a data structure that maps keys to values using a hash function. This page covers how hash tables work, dictionary operations in Python, and handling collisions.

## Overview

Hash tables provide average O(1) time complexity for insertions, deletions, and lookups, making them one of the most efficient data structures for key-value storage. Python's built-in `dict` is implemented as a hash table.

## Complexity

| Operation | Average | Worst Case |
|-----------|---------|------------|
| Insert    | O(1)    | O(n)       |
| Lookup    | O(1)    | O(n)       |
| Delete    | O(1)    | O(n)       |
| Space     | -       | O(n)       |
