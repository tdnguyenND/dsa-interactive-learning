# Hash Map Patterns

Hash maps enable $O(1)$ average-case lookups, making them the go-to data structure for a wide range of algorithmic patterns. This page covers the most common hash map patterns you will encounter in coding interviews.

## Pattern 1: Frequency Counting

**When to use**: Count occurrences of elements, find the most/least frequent, check if two collections have the same distribution.

**Template**: Build a frequency map, then query it.

<CodePlayground
  code={`from collections import Counter

def top_k_frequent(nums, k):
    """Find the k most frequent elements.
    Time: O(n), Space: O(n) — uses bucket sort by frequency"""
    count = Counter(nums)
    n = len(nums)
    buckets = [[] for _ in range(n + 1)]
    for num, freq in count.items():
        buckets[freq].append(num)

    result = []
    for freq in range(n, 0, -1):
        for num in buckets[freq]:
            result.append(num)
            if len(result) == k:
                return result
    return result

print(top_k_frequent([1,1,1,2,2,3], 2))  # [1, 2]
print(top_k_frequent([1], 1))             # [1]`}

<CodePlayground

    """Check if t is an anagram of s.
    Time: O(n), Space: O(1) — at most 26 letters"""
    if len(s) != len(t):
        return False
    freq = {}
    for c in s:
        freq[c] = freq.get(c, 0) + 1
    for c in t:
        freq[c] = freq.get(c, 0) - 1
        if freq[c] < 0:

    return True

    Time: O(n), Space: O(1)"""

    for i, c in enumerate(s):
        if freq[c] == 1:
            return i
    return -1

print(first_unique_char("aabb"))         # -1`}

## Pattern 2: Complement Lookup (Two Sum Pattern)

**When to use**: Find pairs/groups that satisfy a condition. Instead of checking all pairs in $O(n^2)$, store seen elements and look up the complement in $O(1)$.

**Key insight**: For each element $x$, check if `target - x` is already in the map.

<CodePlayground
  code={`def two_sum(nums, target):
    """Find indices of two numbers that sum to target.
    Time: O(n), Space: O(n)"""
    seen = {}  # value -> index
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
    return []

print(two_sum([2, 7, 11, 15], 9))   # [0, 1]
print(two_sum([3, 2, 4], 6))        # [1, 2]

def two_sum_count(nums, target):
    """Count pairs that sum to target.
    Time: O(n), Space: O(n)"""
    seen = {}
    count = 0
    for num in nums:
        complement = target - num
        if complement in seen:
            count += seen[complement]
        seen[num] = seen.get(num, 0) + 1
    return count

print(two_sum_count([1, 5, 7, -1, 5], 6))  # 3 (1+5, 1+5, 7+-1)
print(two_sum_count([1, 1, 1, 1], 2))       # 6`}
  title="Two Sum and Complement Lookup"

## Pattern 3: Grouping

**When to use**: Group elements by some shared property (e.g., anagrams, same remainder, same pattern).

**Template**: Compute a **hashable key** for each element and collect elements by key.

<CodePlayground
  code={`from collections import defaultdict

def group_anagrams(strs):
    """Group strings that are anagrams of each other.
    Key: sorted string (or character frequency tuple).
    Time: O(n * k log k) where k = max string length"""
    groups = defaultdict(list)
    for s in strs:
        key = tuple(sorted(s))
        groups[key].append(s)
    return list(groups.values())

result = group_anagrams(["eat","tea","tan","ate","nat","bat"])
for group in result:
    print(sorted(group))

print()

def group_anagrams_fast(strs):
    """O(n * k) version using character count as key."""
    groups = defaultdict(list)
    for s in strs:
        count = [0] * 26
        for c in s:
            count[ord(c) - ord('a')] += 1
        groups[tuple(count)].append(s)
    return list(groups.values())

result = group_anagrams_fast(["eat","tea","tan","ate","nat","bat"])
for group in result:
    print(sorted(group))`}
  title="Group Anagrams"

## Pattern 4: Prefix Sum + Hash Map

**When to use**: Find subarrays with a given sum, count subarrays with property X. The hash map stores prefix sums seen so far.

**Key insight**: If $\text{prefix}[j] - \text{prefix}[i] = k$, then the subarray from index $i+1$ to $j$ sums to $k$. So for each prefix sum, check if `prefix_sum - k` was seen before.

<CodePlayground
  code={`def subarray_sum_count(nums, k):
    """Count subarrays that sum to k.
    Time: O(n), Space: O(n)"""
    count = 0
    prefix = 0
    seen = {0: 1}  # prefix sum 0 occurs once (empty prefix)

    for num in nums:
        prefix += num
        # If prefix - k was seen, those positions are valid starts
        if prefix - k in seen:
            count += seen[prefix - k]
        seen[prefix] = seen.get(prefix, 0) + 1

    return count

print(subarray_sum_count([1, 1, 1], 2))        # 2
print(subarray_sum_count([1, 2, 3], 3))         # 2 ([1,2] and [3])
print(subarray_sum_count([1, -1, 0], 0))        # 3`}
  title="Subarray Sum Equals K"
/>

<CodePlayground
  code={`def longest_subarray_sum_k(nums, k):
    """Find the length of the longest subarray summing to k.
    Store the FIRST occurrence of each prefix sum."""
    prefix = 0
    first_seen = {0: -1}  # prefix 0 at index -1
    max_len = 0

    for i, num in enumerate(nums):
        prefix += num
        if prefix - k in first_seen:
            max_len = max(max_len, i - first_seen[prefix - k])
        # Only store first occurrence for longest subarray
        if prefix not in first_seen:
            first_seen[prefix] = i

    return max_len

print(longest_subarray_sum_k([1, -1, 5, -2, 3], 3))   # 4 ([1,-1,5,-2])
print(longest_subarray_sum_k([-2, -1, 2, 1], 1))       # 2

def subarrays_div_by_k(nums, k):
    """Count subarrays whose sum is divisible by k.
    Key: if two prefix sums have the same remainder mod k,
    the subarray between them is divisible by k."""
    count = 0
    prefix = 0
    remainder_count = {0: 1}

    for num in nums:
        prefix += num
        r = prefix % k
        if r in remainder_count:
            count += remainder_count[r]
        remainder_count[r] = remainder_count.get(r, 0) + 1

    return count

print(subarrays_div_by_k([4, 5, 0, -2, -3, 1], 5))  # 7`}

## Pattern 5: Sliding Window + Hash Map

**When to use**: Find substrings/subarrays satisfying a frequency-based condition (e.g., no repeating characters, at most $k$ distinct characters, contains all characters of a target).

**Template**: Expand the window by moving the right pointer, update the hash map, shrink from the left when the condition is violated.

<CodePlayground
  code={`def length_of_longest_substring(s):
    """Longest substring without repeating characters.
    Time: O(n), Space: O(min(n, alphabet_size))"""
    last_seen = {}  # char -> last index
    max_len = 0
    left = 0

    for right, ch in enumerate(s):
        if ch in last_seen and last_seen[ch] >= left:
            left = last_seen[ch] + 1
        last_seen[ch] = right
        max_len = max(max_len, right - left + 1)

    return max_len

print(length_of_longest_substring("abcabcbb"))  # 3 ("abc")
print(length_of_longest_substring("bbbbb"))      # 1 ("b")
print(length_of_longest_substring("pwwkew"))     # 3 ("wke")`}
  title="Longest Substring Without Repeating Characters"
/>

<CodePlayground
  code={`def min_window(s, t):
    """Minimum window substring containing all chars of t.
    Time: O(n), Space: O(k) where k = distinct chars in t"""
    from collections import Counter
    need = Counter(t)
    missing = len(t)  # total chars still needed
    left = 0
    best = (0, float('inf'))  # (start, end)

    for right, ch in enumerate(s):
        if need[ch] > 0:
            missing -= 1
        need[ch] -= 1

        # Shrink window while valid
        while missing == 0:
            if right - left < best[1] - best[0]:
                best = (left, right)
            need[s[left]] += 1
            if need[s[left]] > 0:
                missing += 1
            left += 1

    start, end = best
    return s[start:end + 1] if end < float('inf') else ""

print(min_window("ADOBECODEBANC", "ABC"))  # "BANC"
print(min_window("a", "a"))                # "a"
print(min_window("a", "aa"))               # ""`}
  title="Minimum Window Substring"
/>

<CodePlayground
  code={`def longest_k_distinct(s, k):
    """Longest substring with at most k distinct characters.
    Time: O(n), Space: O(k)"""
    freq = {}
    left = 0
    max_len = 0

    for right, ch in enumerate(s):
        freq[ch] = freq.get(ch, 0) + 1

        while len(freq) > k:
            left_ch = s[left]
            freq[left_ch] -= 1
            if freq[left_ch] == 0:
                del freq[left_ch]
            left += 1

        max_len = max(max_len, right - left + 1)

    return max_len

print(longest_k_distinct("eceba", 2))        # 3 ("ece")
print(longest_k_distinct("aabacbebebe", 3))  # 7 ("cbebebe")`}
  title="Longest Substring with K Distinct Characters"

## Pattern comparison

| Pattern | Hash Map Role | Typical Time | Typical Space |
|---------|---------------|-------------|---------------|
| **Frequency counting** | Count occurrences | $O(n)$ | $O(k)$ |
| **Complement lookup** | Store seen values for $O(1)$ pair check | $O(n)$ | $O(n)$ |
| **Grouping** | Collect elements by computed key | $O(n \cdot f)$ | $O(n)$ |
| **Prefix sum + hash** | Store prefix sums to find subarray sums | $O(n)$ | $O(n)$ |
| **Sliding window + hash** | Track frequencies in current window | $O(n)$ | $O(k)$ |

Where $k$ is the number of distinct keys and $f$ is the cost of computing the grouping key.

## How to choose the right pattern

| Signal in the problem | Pattern |
|----------------------|---------|
| "Count occurrences" or "most/least frequent" | **Frequency counting** |
| "Find pair/tuple that sums to X" | **Complement lookup** |
| "Group by property" or "find all anagrams" | **Grouping** |
| "Count/find subarrays with sum = k" | **Prefix sum + hash map** |
| "Longest substring with condition" | **Sliding window + hash map** |
| "Check if permutation exists in string" | **Sliding window + hash map** |

## Common mistakes

1. **Forgetting to initialize prefix sum map with `{0: 1}`** --- this accounts for subarrays starting at index 0.
2. **Using a mutable key** (like a list) in a hash map --- use tuples or frozensets instead.
3. **Not handling negative numbers** in prefix sum problems --- remainders can be negative in some languages (Python handles this correctly with `%`).
4. **Off-by-one in sliding window** --- make sure `left` is updated correctly when shrinking.
5. **Storing last occurrence instead of first** when finding the longest subarray --- for longest, store the first occurrence; for shortest, store the last.

## Key takeaways

1. **Frequency counting** is the simplest pattern: build a counter, then query it. Use `collections.Counter` in Python.
2. **Complement lookup** (Two Sum pattern) reduces $O(n^2)$ pair search to $O(n)$ by storing seen values and checking for complements.
3. **Grouping** requires designing a **hashable key** that captures the shared property --- sorted string, frequency tuple, or remainder.
4. **Prefix sum + hash map** is the key technique for subarray sum problems: store prefix sums and look up `prefix - target`.
5. **Sliding window + hash map** handles substring/subarray problems with frequency constraints: expand right, shrink left, track frequencies in the map.
6. When you see "find subarray/substring with property X" --- think prefix sum + hash map for sum-based properties, or sliding window + hash map for frequency-based constraints.
