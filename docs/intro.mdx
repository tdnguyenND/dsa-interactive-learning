# DSA Interactive Learning

Welcome to **DSA Interactive Learning** — a comprehensive, interactive guide to Data Structures and Algorithms.

## What makes this different?

- **Run code in your browser** — every topic includes a Python code editor powered by Pyodide (CPython compiled to WebAssembly). Write, modify, and execute code without any setup.
- **Step-by-step visualizations** — watch algorithms execute with play/pause controls, speed adjustment, and detailed explanations at each step.
- **Comprehensive coverage** — from basic arrays to dynamic programming, organized as a structured curriculum.

## How to use this site

1. **Follow the sidebar** — topics are ordered from fundamentals to advanced. Each section builds on the previous one.
2. **Read the theory** — understand the concept, its properties, and when to use it.
3. **Watch the visualization** — use the algorithm visualizer to see how operations work step by step.
4. **Run the code** — modify the code examples and run them to experiment.
5. **Check complexity** — every algorithm includes time and space complexity analysis.

## Prerequisites

- Basic programming knowledge (variables, loops, functions)
- Familiarity with Python syntax (all code examples use Python)
- No prior DSA knowledge required — we start from the fundamentals
